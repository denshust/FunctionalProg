Мета

Завдання 1

Завдання 6.1 Створити вектор чисел. Вивести на екран елементи, які менші за значення максимального і більше за значення мінімального елементів.


Хід роботи

  • Вектори – це різнорідні структури, елементи яких індексуються цілими числами. Довжина вектора – це кількість елементів, які він містить. Це число є невід’ємним цілим числом, яке фіксується при створенні вектора.

  • Індекси вектора – це точні невід’ємні цілі числа, менші за довжину вектора. Перший елемент у векторі індексується нулем, останній елемент індексується на одиницю менше довжини вектора.

  • В мові Scheme (Racket) вектори записуються із позначенням # (obj ...). Наприклад, вектор довжиною 3, що містить нульове значення в індексі 0, список (2 2 2 2) в індексі 1 та рядок "Аnnа" в індексі 2, можна записати так: #(0 (2 2 2 2) "Anna"). Векторні константи повинні quoted (цитуватися) за допомогою символу ‘, наприклад,

  • ‘ #(0 (2 2 2 2) "Anna") , що означає # (0 (2 2 2 2) "Аnnа").


Код програми



let array = [| 1; 2; 3; 4; 5|]

printfn "array: %A" array

printfn ""

let mutable min = array |> Array.min

let mutable max = array |> Array.max

let array1 = Array.choose (fun elem -> if not(elem = min) && not(elem = max) then

Some(int (elem))

else

None) array

printfn "array1: %A "array1

printfn ""




Завдання 2


Завдання 6.2 Створити дві черги з різною кількістю елементів. Об’єднати черги, включивши в об’єднану чергу однакову кількість елементів з кожної вхідної черги.


Код програми


let mutable array1 = [| 1; 2; 3; 4; 5|]

printfn "array1: %A "array1

let mutable array2 = [| 1; 2; 3; 4; 5; 6|]

printfn "array2: %A "array2

let mutable array3 = Array.append array1 array2

let mutable (array4 : int array) = Array.zeroCreate (array3.Length-1)

for i in 0 .. array3.Length - 2 do

array4.[i] <- array3.[i]

printfn "array4: %A "array4
