def countChange(money: Int, coins: List[Int]): Int =

if (money == 0) 1

else if (money < 0) 0

else if (coins.isEmpty) 0

else countChange(money - coins.head, coins) + countChange(money, coins.tail)




Результати

Враховуючи значення N, якщо ми хочемо дати решту на N центів, і ми маємо нескінченну кількість кожної з монет, що мають значення S = {S1, S2, .., Sm} та порядок монет не має значення, то :

Наприклад, для N = 4 та S = {1,2,3} є чотири рішення: {1,1,1,1}, {1,1,2}, {2,2}, {1, 3}. Отже, вихід повинен бути 4.

Для N = 10 і S = {2, 5, 3, 6} існує п’ять рішень: {2,2,2,2,2}, {2,2,3,3}, {2,2,6}, {2,3,5} та {5,5}. Отже, вихід повинен бути 5.

Для N = 10 і S = {30, 5, 7} існує лише одне рішення : {5,5}. Отже, вихід повинен бути 1.
